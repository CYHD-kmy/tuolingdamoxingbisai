# 智投未来 — 运行指南

## 这是什么？

一个 A 股投资 AI 系统。每天自动扫描 5000+ 股票，用 AI 分析后选出当天最值得投资的标的，输出标准 JSON 格式的投资建议。

**运行它只需要一条命令。**

---

## 第一次运行（3 步）

### 第 1 步：安装依赖

打开终端，进入项目目录，运行：

```bash
pip install -r requirements.txt
```

这一步只需做一次。安装过程约 2-3 分钟。

### 第 2 步：配置

```bash
python manage.py setup
```

会弹出交互式提示，依次输入：

```
1. DeepSeek API Key →  粘贴你的 Key（必填）
2. 模型配置         →  直接回车（用默认值）
3. Tushare Token    →  直接回车跳过，或粘贴 Token（可选）
```

**没有 DeepSeek API Key？** 去 [platform.deepseek.com/api_keys](https://platform.deepseek.com/api_keys) 免费注册获取。

> 如果暂时没有 Key，也可以跳过第 2 步，直接用演示模式体验（见下方）。

### 第 3 步：运行

```bash
# 正常模式（需要 API Key）
python manage.py run

# 演示模式（不需要 API Key，用模拟数据体验）
python manage.py run-demo
```

运行时间：演示模式约 15 秒，正常模式约 3-5 分钟。

## 每日自动化运行

如果你希望系统每天自动在开盘前运行，不需要手动操作：

```bash
python manage.py schedule
```

这会在后台启动一个定时任务，每个交易日 9:00 自动执行。结果保存在 `results/` 目录。

```bash
# 停止定时任务
python manage.py schedule-stop

# 查看是否在运行
python manage.py status
```

## 常见问题

### Q: 需要什么操作系统？
A: macOS / Linux / Windows 都可以。Python 3.10+ 即可。

### Q: 需要一直开着电脑吗？
A: 定时模式（`schedule`）需要电脑开机。如果想关机也能跑，可以考虑部署到云服务器——见 `DEPLOY.md`。

### Q: 运行失败怎么办？
A: 查看日志文件：

```bash
tail -50 logs/schedule.log    # 定时任务日志
tail -50 logs/server.log      # Web 看板日志
```

常见原因：
- **网络问题**：数据源暂时不可用，系统会自动重试 3 次
- **API Key 无效**：运行 `python manage.py setup` 重新配置
- **非交易日**：周末和节假日自动跳过

### Q: 输出为什么是空数组 [] ？
A: 这意味着当天没有符合条件的投资标的。系统判断市场风险较大或数据不完整，选择空仓——这是风控在起作用，不是出错。

### Q: 能改投资本金吗？
A: 编辑 `.env` 文件，修改对应参数即可。默认是 50 万元。

### Q: .env 文件在哪里？
A: 项目根目录下。可以用任何文本编辑器打开。
